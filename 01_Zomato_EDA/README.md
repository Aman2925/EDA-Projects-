# Zomato Exploratory Data Analysis

## Project Overview
This project involves performing Exploratory Data Analysis (EDA) on the Zomato dataset to analyze country-wise restaurant distribution, customer rating patterns, engagement levels, and currency usage.

## Objectives
- Understand the structure and key attributes of the dataset
- Analyze country-wise restaurant distribution
- Examine customer rating patterns and engagement
- Identify countries with zero ratings
- Study currency usage across different countries
- Derive actionable business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- India dominates the dataset with the highest number of records
- Most restaurants fall under Good to Excellent rating categories
- Higher ratings are associated with higher customer engagement
- Certain countries show zero ratings indicating low engagement
- Currency usage is country-specific
